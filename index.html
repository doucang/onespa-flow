◇ 抽成准入：履约完成 且 未退款？
↓
是进抽成池
可主张抽成订单池仅含「已履约 且 未退款」订单 · 不含全店 walk-in
否剔除
不参加抽成已付款但未履约 / 已退款 —— 哪怕钱已收，也不算
⊖ 退款扣除：已入池订单若事后退款 → 从抽成基数扣回 ↓
按周 / 月生成抽成结算报表基数 = 已履约 且 未退款 · 退款逐笔扣除
可抽成金额 = 归属小ONE渠道 + 已履约 + 未退款 的净金额
部分退款 → 净额 = 付款金额 − 退款金额　|　全额退款 → 0　|　已付款但未履约 → 0
↓
与 One Spa 对账结算ERP 为财务底；小ONE 抽成池为主张证据；BK 区每周并入对账
关键规则说明
  • 小ONE ≠ ERP小ONE 是订单入口 / 入库 / 通知 / 抽成证据；ERP 才是财务系统。
  • 预约总账 ≠ 最终财务系统预约总账只是前台准备与到店标绿工具。
  • 打单 = 入账前台打单 → 正式进入 One Spa ERP / 收银。
  • 标绿 = 到店核销= 已到店 / 已履约，不等于财务入账。
  • 抽成 = 履约 且 未退款付款只是必要条件，不是充分条件。
  • 付款 ≠ 可抽成未履约 / 已退款一律剔除；退款从基数扣回。
  • 可抽成 = 履约 + 未退款 的净额台账三轴：付款 / 履约 / 退款。部分退款扣退款额，全额退款记 0。
  • 已付款订单付款后马上打单，不等客户到店。
  • 未付款订单客户到店收款后，才打单入账。
  • BK 区 = 线下买单 · 旁路 · 每周对账不进散客群实时链路；外派也不进预约总账，履约均回写小ONE 台账。
  • 小ONE 台账只记可主张抽成订单证据不记录全店所有 walk-in。
